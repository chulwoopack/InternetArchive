# Internet Archive

A python and command-line interface to Archive.org

## Getting Started

* How to setup
```
https://github.com/jjjake/internetarchive
```

* After setup, setup a configuration
```
$ ia configure
```

* Ready to go! Download the file with searching
```
$ ia download --search 'newspaper'
```
